Projektbeschreibung

Dieses Projekt generiert automatisch Flashcards aus englisch-deutschen Wortlisten mit Hilfe des lokalen KI-Modells Mistral über Ollama.

Features

  • Automatische Umwandlung von Text in ein strukturiertes JSON-Format.
  • Verwendung von Ollama, um den Text semantisch zu verstehen.
  • Erstellung von Karteikarten mit englischen Begriffen, Beispielsätzen und deutschen Übersetzungen.

Beispiel-Screenshots

Code


import subprocess
import json

def format_text_with_ollama(text):
    prompt = f"""
    Formatiere den Text als Python-Liste mit:
    [["Englisches Wort", "Beispielsatz", "Deutsche Übersetzung"], ...].
    
    Text:
    {text}
    """
    result = subprocess.run(
        ["ollama", "run", "mistral", prompt], 
        capture_output=True, text=True, encoding='utf-8'
    )
    return result.stdout.strip()
