Aktuelle Projekte

SmartScale

SmartScale ist eine intelligente Waage, die Rezepte über eine Webanwendung empfängt.
Nutzer können Rezepte in eine Datenbank hinzufügen oder löschen, wobei jedes Rezept Details wie Schärfegrad, Zubereitungsdauer, Zutatenliste und ein Profilbild enthält. Die Webanwendung ermöglicht außerdem das automatische Erstellen von Einkaufslisten basierend auf ausgewählten Rezepten. Zusätzlich verfügt die Waage über ein Touch-Display, das den Nutzer Schritt für Schritt durch das Rezept führt und die benötigten Mengen direkt abwiegt. Erste Prototypen wurden bereits erfolgreich getestet. Link: zur Website von SmartScale
